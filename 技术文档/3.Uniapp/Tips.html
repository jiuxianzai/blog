解除样式隔离 ​

WARNING

  • 由于微信存在组件样式隔离。要么在全局 scss 去写要么配置。
  • 微信小程序可以添加组件样式，而支付宝不支持。建议套层view标签写类名
vue
<script>
export default {
  options: {
    styleIsolation: 'shared',
  },
};
</script>
